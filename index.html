📱 ¿Cómo funciona el registro?
Opción 1 — Este chat (lo más fácil): Escribe cualquier movimiento en lenguaje natural. El tracker lo interpreta, lo clasifica y actualiza tus saldos automáticamente. Ej: "gasté 120 soles en el super" o "recibí $2,680 de salario".

Opción 2 — Registro manual: Usa el formulario en la pestaña Transacciones para mayor control — puedes elegir cuenta, categoría, fecha y estado exactos.

💾 Respaldo: Tus datos viven en este navegador. Ve a Configuración → Exportar JSON regularmente para guardar una copia. Si cambias de dispositivo, importa el JSON para restaurar todo.
